# IT Support Hub

Sistema moderno de soporte técnico con gestión de tickets y dashboard en tiempo real.

## Características

- Dashboard con métricas en tiempo real
- Sistema de tickets con vista Kanban
- Vista de lista de tickets
- Diseño moderno con Tailwind CSS
- Totalmente responsive

## Tecnologías

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Lucide Icons

## Inicio rápido

```bash
# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run dev

# Construir para producción
npm run build

# Iniciar en producción
npm run start
```
